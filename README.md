### Лабораторная работа №2: Стек, Дек, Очередь
Реализация линейных структур данных на основе динамических массивов или списков:
- **Стек (Stack):** Принцип LIFO (Last-In, First-Out). Реализованы методы `push`, `pop`, `top`.
- **Очередь (Queue):** Принцип FIFO (First-In, First-Out). Реализованы методы `enqueue`, `dequeue`, `front`.
- **Дек (Deque):** Структура, позволяющая вставлять и удалять элементы с обоих концов.

### Лабораторная работа №3: Хеш-таблица
Реализация ассоциативного контейнера "хеш-таблица":
- Хеш-функция для преобразования ключа в индекс.
- Разрешение коллизий
- Базовые операции: вставка, удаление и поиск элемента по ключу.
- Динамическое расширение таблицы (rehashing).

### Лабораторная работа №4: Двоичные деревья поиска
Работа с иерархическими структурами данных. Реализованы два вида деревьев:
1.  **Обычное бинарное дерево поиска (BST):**
    - Вставка, удаление и поиск узлов.
    - Обходы дерева (прямой, симметричный, концевой).
    - Поиск минимального/максимального элемента.
2.  **Сбалансированное левостороннее красно-черное дерево (Left-leaning Red-Black Tree):**
    - Реализация сбалансированной версии двоичного дерева.
    - Поддержание баланса при вставке и удалении с помощью вращений и перекрашивания узлов.
    - Гарантия того, что длина пути от корня до листьев отличается не более чем в 2 раза.

### Лабораторная работа №5: Графы
Исследование алгоритмов на графах:
- Представление графа (матрица смежности или списки смежности).
- **Поиск в глубину (DFS):** Обход графа, поиск компонент связности.
- **Поиск в ширину (BFS):** Поиск кратчайшего пути в невзвешенном графе.
- (Возможно) Алгоритмы нахождения минимального остовного дерева (Прима) или кратчайших путей (Дейкстры).
